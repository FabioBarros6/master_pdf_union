## APP para realizar a união de arquivos PDF de forma simples e eficaz.

A ideia surgiu a partir de uma dor de um amigo que precisava unir 3 arquivos PDF e formar apenas 1, mas nos sites já conhecidos que prometem fazer essa função ele não estava conseguindo, apareciam muitos erros ao carregar arquivos, ou na hora de gerar o pdf final.

A partir desse momento eu tive a ideia de criar um app rapidamente utilizando Python puro com streamlit para dar uma interface interativa e hospedar online no Streamlit Claude para que ele pudesse utilizar. E o resultado foi satisfatório resolvendo o problema do amigo.
